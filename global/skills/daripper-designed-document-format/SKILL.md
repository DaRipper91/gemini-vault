# 🌌 DaRipper Designed Document Format (D3F) 🌌

This skill activates a high-fidelity chain of four specialized agents to transform any source document into an authoritative, 6-part Master Handbook.

## 🧊 THE AGENT CHAIN
1. **Architect:** Maps every detail from the target file and related system skills.
2. **SME:** Extracts every command, manifest, and skill content WITHOUT sampling.
3. **Auditor:** Executes a 20-Point Editorial Audit (Zero Slop, Zero Hallucinations).
4. **UX Writer:** Applies a Dark Pastel Theme with logos, badges, and expert callouts.

## 🦄 THE 5-PHASE WORKFLOW
1. **Context & Discovery:** Analyze target file for core concepts and technical specs.
2. **Task Definition:** Map facts to the 6-part handbook structure.
3. **Execution (Drafting):** Draft content based strictly on source truth.
4. **Review & Refine (The 20-Point Audit):** Eliminate passive voice, redundant phrasing, and "AI slop."
5. **Iteration & Polish (Callouts):** Inject Dark Pastel styling and expert callouts.

## 👾 REQUIRED DOCUMENT STRUCTURE
1. **User Manual:** High-level overview, quick start, and basic operations.
2. **Technical Manual:** System architecture, infrastructure, and configuration.
3. **Developer Guide:** Code structure, contribution guidelines, and testing.
4. **Standard Operating Procedures (SOP):** Step-by-step instructions for core workflows.
5. **Glossary:** Definition of key terms.
6. **Reference Guide:** Command lists and quick-lookup tables.

## 🌌 STYLING & CALLOUTS
- **Aesthetic:** Dark Pastel (Emojis: 🌌, 🧊, 🦄, 👾).
- **Formatting:** Heavy use of bolding, badges, and logos.
- **Expert Callouts:**
  - `> 💡 **Pro Tip / Expert Opinion:** [Context-aware technical suggestion]`
  - `> ⚠️ **Watch Out:** [Warning about potential pitfalls]`
  - `> 🧠 **Architectural Note:** [Insight on system design reasoning]`

## ⚠️ DIRECTIVE: ZERO HALLUCINATIONS
You are strictly forbidden from inventing facts. If information is missing for a section, you must explicitly state: "> ⚠️ **Data Not Provided in Source Document.**"
